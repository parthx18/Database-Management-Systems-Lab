DDL : DATA DEFINITION LANGUAGE
IMPLEMENTING BASIC SQL COMMANDS
1.CREATE : TO CREATE TABLES, PROCEDURES, TRIGGERS ETC
2.ALTER : TO MODIFY THE STRUCTURE OF TABLE 
3.TRUNCATE : TO DELETE ALL THE DATA, BUT NOT THE TABLE
4.DROP : ENTIRE TABLE WILL BE DELETED, ALSO DATA
5.RENAME : RENAME TABLES, COLUMNS IN DBMS
Example : Alter table St_info Rename to St_information
for columns : Alter table St_information Rename m1 to physics 
NOTE : This is auto commit language.(everything will be saved.)

DML : DATA MANIPULATION LANGUAGE
IMPLEMENTING BASIC SQL COMMANDS
1.INSERT COMMAND : TO INSERT DATA INTO THE TABLE
2.UPDATE : TO UPDATE THE COLUMNS (USE SET WITH IT)
